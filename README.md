### Приложение для получения погоды через терминал .
1. Создается список с локациями, в которых необходимо получить данные.
2. Делается запрос к сайту wttr.in
3. В терминале выводится погода в запрашиваемых местах на три дня.
````

       `-’      10 км          
      /   \     0.0 мм
````
